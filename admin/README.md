# PROLOG Admin Web

관리자 전용 웹 대시보드

## 예정 기능

- 사용자 관리
- 운동 종목 관리 (시스템 종목 CRUD)
- 통계 대시보드
- 신고 관리 (Phase 3 커뮤니티 이후)

## 기술 스택 (예정)

- **Framework:** Next.js
- **UI:** shadcn/ui + Tailwind CSS
- **Deploy:** Vercel
