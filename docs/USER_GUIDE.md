## 1. PROLOG란?

PROLOG는 웨이트 트레이닝 기록과 성장 추적에 특화된 운동 앱입니다.

**핵심 기능:**
- **루틴 기반 운동** — 자주 하는 운동 조합을 루틴으로 저장해두고 반복 사용
- **이전 기록 자동 표시** — 같은 루틴으로 운동할 때 지난번 무게/횟수를 바로 확인
- **성장 추적** — 이번 주/달 운동 횟수, 자주 하는 운동 TOP 5 등 통계 제공
- **커뮤니티** — 루틴을 공유하고, 다른 사람의 루틴을 가져와서 바로 사용

## 2. 시작하기

### 회원가입

1. 앱 실행 후 **회원가입** 탭 선택
2. 아이디(영문/숫자), 비밀번호, 이메일, 닉네임 입력
3. 가입 완료 후 자동 로그인

### 프로필 설정

하단 탭 **프로필 → 프로필 수정**에서 설정 가능:
- 닉네임
- 성별
- 키 / 체중

> 신체 정보는 선택 사항이며, 나중에 언제든 수정할 수 있습니다.

## 3. 루틴 관리

루틴은 자주 하는 운동들의 묶음입니다. 예: "상체 루틴 A", "하체 데이"

### 루틴 만들기

1. 하단 탭 **루틴** 선택
2. 우측 상단 **+** 버튼
3. 루틴 이름, 설명(선택) 입력
4. 운동 종목 추가
   - 부위별로 필터링해서 종목 선택
   - 종목마다 **목표 세트 수**와 **휴식 시간** 설정
5. **저장**

### 루틴 수정

루틴 상세 페이지에서 **수정** 버튼 → 종목 추가/삭제, 순서 변경, 세트 수 조정 가능

> 루틴을 수정해도 이전에 기록된 운동 내역은 변경되지 않습니다.

### 루틴 보관 / 삭제

더 이상 사용하지 않는 루틴은 **보관** 처리하면 목록에서 숨겨집니다.

- **활성 루틴** → 보관 버튼으로 보관 처리
- **보관된 루틴** → 다시 활성화하거나 완전히 삭제 가능

> 삭제는 보관 상태에서만 가능합니다. 실수 방지를 위한 2단계 프로세스예요.

## 4. 운동 기록하기

### 운동 시작

하단 탭 중앙의 **덤벨(🏋️) 버튼** 또는 운동 탭에서 시작:

- **루틴으로 시작** — 저장한 루틴 선택 후 시작
- **자유 운동** — 루틴 없이 그때그때 종목 선택

### 운동 화면 구성

- 진행 시간이 상단에 실시간으로 표시됩니다
- 각 종목마다 **이전 기록이 자동으로 표시**됩니다 (같은 루틴 기준)
- 세트별로 무게(kg)와 횟수(reps) 입력 후 체크 ✓

### 운동 중 종목 추가

운동 화면 하단 **종목 추가** 버튼으로 진행 중에도 종목을 추가할 수 있습니다.

### 운동 완료

완료 버튼을 누르면 4가지 저장 방식 중 선택:

| 옵션 | 설명 | 언제 사용? |
|------|------|-----------|
| **기록만 저장** | 루틴 변경 없이 기록만 남김 | 가장 일반적인 경우 |
| **루틴으로 만들기** | 오늘 한 운동을 새 루틴으로 저장 | 자유 운동 후 루틴화할 때 |
| **자유 운동으로 전환** | 루틴 연결 끊고 자유 운동으로 기록 | 루틴과 다르게 운동했을 때 |
| **루틴 업데이트** | 오늘 한 대로 루틴 구성도 변경 | 루틴을 개선하고 싶을 때 |

### 운동 취소

운동하다가 그만두고 싶을 때 **취소** 버튼 → 기록이 저장되지 않습니다.

## 5. 홈 화면 통계

홈 탭에서 확인할 수 있는 정보:

| 항목 | 내용 |
|------|------|
| **이번 주** | 이번 주 운동 횟수 |
| **이번 달** | 이번 달 운동 횟수 |
| **평균 운동 시간** | 세션당 평균 소요 시간 |
| **주간 활동** | 최근 7일 운동 기록 (어떤 부위를 했는지) |
| **자주 하는 운동 TOP 5** | 가장 많이 기록한 종목 |
| **최근 운동 기록** | 최근 완료한 세션 목록 |

## 6. 커뮤니티

### 루틴 둘러보기

하단 탭 **커뮤니티**에서 다른 사용자들이 공유한 루틴을 볼 수 있습니다.

- **최신순 / 인기순 / 많이 가져간 순** 정렬 가능
- 각 카드에서 운동 부위, 종목 수, 조회수 확인

### 루틴 가져오기

마음에 드는 루틴을 내 루틴으로 가져올 수 있습니다.

1. 공유 루틴 상세 페이지 접속
2. **가져오기** 버튼
3. 내 루틴 목록에 자동 추가됩니다

> 가져온 루틴은 내 것이 되어 자유롭게 수정할 수 있습니다.
> 이미 가져온 루틴은 가져오기 버튼이 비활성화됩니다.

### 내 루틴 공유하기

1. 커뮤니티 탭 또는 루틴 상세에서 **공유하기** 버튼
2. 공유 제목과 설명 작성 (원본 루틴 이름과 다르게 써도 됩니다)
3. 공유 완료

> 공유 후 원본 루틴을 수정하거나 삭제해도 공유된 내용은 유지됩니다.

### 댓글

공유 루틴 상세 페이지에서 댓글을 남기거나 확인할 수 있습니다.
본인이 작성한 댓글만 삭제 가능합니다.

## 7. 자주 묻는 질문 (FAQ)

**Q. 운동 기록을 삭제할 수 있나요?**
네. 운동 기록 상세 페이지에서 삭제할 수 있습니다. 삭제하면 복구되지 않습니다.

**Q. 루틴을 삭제하면 이전 운동 기록도 사라지나요?**
아니요. 루틴을 삭제해도 과거에 그 루틴으로 운동한 기록은 그대로 유지됩니다.

**Q. 커스텀 운동 종목을 추가할 수 있나요?**
네. 루틴 편집 화면 또는 운동 중 종목 선택 시 **커스텀 종목 추가** 기능을 사용할 수 있습니다.

**Q. 이전 운동 기록은 어디서 볼 수 있나요?**
하단 탭 **프로필 → 운동 기록**에서 전체 기록을 확인할 수 있습니다.

**Q. 같은 루틴으로 운동할 때 지난번 기록이 안 보여요.**
루틴 기반으로 세션을 시작해야 이전 기록이 표시됩니다. 자유 운동으로 시작한 경우에는 표시되지 않습니다.

**Q. 아이디/비밀번호를 잊어버렸어요.**
현재 버전에서는 비밀번호 재설정 기능이 준비 중입니다.

**Q. 회원 탈퇴하면 어떻게 되나요?**
계정 및 모든 운동 기록, 루틴, 공유한 루틴, 댓글이 영구 삭제됩니다. 복구가 불가능하니 신중하게 진행해주세요.
